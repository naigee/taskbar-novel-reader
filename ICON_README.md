# 图标文件说明

## 需要准备的图标

请准备一个名为 `icon.png` 的图标文件，放在项目根目录（与本文件同一目录）。

### 图标要求：
- **文件名**: icon.png
- **格式**: PNG（支持透明背景）
- **建议尺寸**: 32x32 像素 或 64x64 像素
- **用途**: 系统托盘图标

### 临时方案：

如果你暂时没有合适的图标，可以：

1. **方法一**：从网上下载免费图标
   - 推荐网站：https://www.iconfinder.com
   - 搜索关键词："book", "reading", "novel"

2. **方法二**：使用 Windows 画图工具制作
   - 打开画图工具
   - 创建 32x32 的新图像
   - 画一个简单的图案（比如一本书的形状）
   - 保存为 `icon.png`

3. **方法三**：使用在线图标生成器
   - https://favicon.io
   - 可以生成简单的文字或图形图标

### 注意事项：

⚠️ **没有图标程序也能正常运行！**

只是系统托盘区域不会显示图标，但应用的所有功能都可以正常使用。
你可以在任务栏看到小说内容显示区域，点击即可阅读。

---

## 快速开始

准备好图标后，将 `icon.png` 放在项目根目录，然后运行：

```bash
npm start
```

祝你阅读愉快！📖
