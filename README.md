# 任务栏小说阅读器

一个简洁的 Electron 应用，在 Windows 任务栏显示小说内容，点击或鼠标滚轮滚动切换段落，悬浮窗查看文字。

## ✨ 功能特性

- 📖 **智能分行**：自动按行分割小说内容，每行最多 25 个字
- 💻 **系统托盘集成**：在任务栏右侧托盘区显示图标
- 🖱️ **点击阅读**：左键点击或鼠标滚轮任务栏窗口显示下一行，右键点击托盘图标显示菜单
- 📚 **多小说支持**：自动扫描 `novel/` 文件夹中的所有 TXT 文件
- 💾 **进度保存**：自动保存每本小说的阅读进度，下次启动继续阅读
- 🔀 **小说切换**：通过右键菜单快速切换不同的小说
- 🎨 **简洁界面**：透明窗口设计，不遮挡其他应用
- ⚡ **轻量快速**：基于 Electron，性能优秀
- ⚡ **压缩小说文件**：使用zip打包压缩，读取时解压(1.0.1版本待实现)

## 🚀 快速开始

### 1. 安装依赖

```bash
npm install
```

### 2. 准备图标（可选）

准备一个 `icon.png` 文件（建议 32x32 像素）放在项目根目录。
没有图标程序也能运行，只是系统托盘不显示图标。

### 3. 添加小说

#### 方式一：单本小说
将小说内容保存为 `novel.txt` 文件，放在项目根目录。

#### 方式二：多本小说（推荐）
创建 `novel/` 文件夹，将多个 TXT 文件放入其中：
```
taskbar-novel-reader/
└── novel/
    ├── 小说 1.txt
    ├── 小说 2.txt
    └── 小说 3.txt
```

### 4. 运行应用

```bash
# 正常运行
npm start

# 开发模式（带开发者工具）
npm run dev
```

## 📝 使用说明

### 基本操作

- **左键点击**任务栏窗口 → 显示下一段内容
- **右击任务栏窗口** → 显示上一段内容
- **右键点击**系统托盘图标 → 弹出操作菜单

### 功能说明

- 每次显示约 **25 个汉字**（自动分行）
- 窗口位置：屏幕底部中央偏右
- 窗口尺寸：300x40 像素
- 支持 **多部小说切换**，每本独立保存进度
- 自动保存阅读进度，关闭应用时也会保存

### 右键菜单功能

- 📖 **小说列表**：显示所有可阅读的小说，单选切换
- 🔄 **重置当前进度**：将当前小说进度重置到第一章
- ❌ **退出**：保存进度并退出应用

## ⚙️ 自定义配置

### 修改显示文字数量

编辑 `main.js` 第 66 行：
```javascript
if (trimmedLine.length > 25) {  // 改为其他数字
    for (let i = 0; i < trimmedLine.length; i += 25) {
        novelContent.push(trimmedLine.substring(i, i + 25));  // 同步修改
    }
}
```

### 调整窗口位置

编辑 `main.js` 第 270-271 行：
```javascript
const x = (width - windowWidth) / 2 + 200;  // 水平位置偏移
const y = height - windowHeight + 40;       // 垂直位置偏移
```

### 修改窗口大小

编辑 `main.js` 第 267-268 行：
```javascript
const windowWidth = 300;   // 窗口宽度
const windowHeight = 40;   // 窗口高度
```

## 💡 提示

2. **多小说管理**：使用 `novel/` 文件夹可以管理多本小说

## 🐛 常见问题

### Q: 为什么系统托盘没有图标？
A: 检查项目根目录是否有 `icon.png` 文件，建议尺寸 32x32 像素。

### Q: 如何切换小说？
A: 右键点击系统托盘图标，在菜单中选择想看的小说。

### Q: 进度丢失了怎么办？
A: 检查 `config.json` 文件是否存在且格式正确。

### Q: 窗口显示位置不合适？
A: 修改 `main.js` 中的 `x` 和 `y` 坐标值（见自定义配置章节）。
